# Project Templates

Plantillas de sistemas completos para la Software Factory. Permiten la generación rápida de estructuras predefinidas.

## CRM (Customer Relationship Management)
Sistema orientado a la gestión de clientes y seguimiento comercial.

**Módulos Incluidos:**
- `clientes`: Gestión de base de datos de personas/empresas.
- `contactos`: Registro de interacciones (llamadas, mails).
- `reportes`: Dashboard de métricas comerciales.
- `campañas`: Gestión de comunicaciones masivas.

---

## Sistema de Pedidos y Ventas
Sistema orientado a la comercialización de productos o servicios.

**Módulos Incluidos:**
- `clientes`: Base de datos vinculada a ventas.
- `pedidos`: Gestión del ciclo de vida de la orden.
- `productos`: Catálogo de bienes o servicios.
- `facturación`: Generación de comprobantes y cobros.

---

## Gestión de Atención Premium
Sistema orientado a la fidelización y post-venta.

**Módulos Incluidos:**
- `clientes`: Segmentación y perfiles.
- `cumpleaños`: Automatización de saludos y beneficios.
- `difusión`: Herramientas de envío masivo vía WhatsApp.
- `encuestas`: Recolección de feedback de satisfacción.
